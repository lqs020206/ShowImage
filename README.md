# 图片压缩对比工具

这是一个纯前端的图片压缩对比工具，支持 BPG 和 JPEG 两种压缩格式。用户可以上传图片，设置压缩参数，查看压缩效果和图像质量指标。

## 功能特点

- 支持图片拖拽上传
- 支持 BPG 和 JPEG 压缩格式
- 可调节压缩质量参数
- 实时显示压缩率
- 显示图像质量评价指标（PSNR、MS-SSIM）
- 支持原图与压缩图对比预览

## 使用方法

1. 打开 index.html 文件
2. 将图片拖入指定区域或点击上传
3. 选择压缩格式（BPG/JPEG）
4. 调节压缩质量参数
5. 查看压缩效果和质量指标

## 技术说明

本项目使用纯前端技术实现，主要依赖：
- bpg-encoder.js：用于BPG压缩
- image-quality.js：用于计算图像质量指标
